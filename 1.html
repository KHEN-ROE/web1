HTML

Hypertext Markup Language (HTML) is the standard markup language language for creating web pages and web applications. Web browsers receive or from local storage and render them into multimdedia web pasges. HTML describes the structure of a web page semantically and originally include cues for the appearance of the document.

Html elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, pragraghs, lists, links, quotes and other items. Html elements are delineated by tags, written using angle brackets.
